Pattern Template Guide

Doppler skins are applied to knives using a universal pattern template. This guide will help explain why the same phase Dopplers will still look different from each other.

The Doppler pattern template shown above is used for all Doppler knives. The position on the x and y axis determines how much of each color (pink, black, blue, etc.) will appear on your blade.

Every skin in Counter-Strike has a pattern template number that is randomly generated and can be 0â€“999. This number determines where the texture file of the skin is applied to each item. In the case of Dopplers, this affects how much of a certain color appears on a knife. For example, some Phase 2 Dopplers will have the entire knife pink, like in the video above. Most Phase 2 Dopplers will have a mix of pink and brown, and these rare all-pink versions are more valuable than your standard Doppler. If you want to explore the 1000 different possibilities of each Doppler, you can use the Pattern Wiki: pattern.wiki.
